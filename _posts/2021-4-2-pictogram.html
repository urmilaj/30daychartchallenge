# Rice Varieties

# Week 1: Comparisons. Day 2: Pictogram.


Day two's topic was pictogram. A pictogram is usage of an icon or symbol to visualize data. I came across this wikipedia page (opens new window) of volunteer sourced list of rice cultivars with each cultivar's name. I decided to visualize this information for day two. Since the information is volunteer sourced, the chart may not be an exhaustive accurate representation of official rice cultivars of each country, since I wanted to practice making a pictogram chart with D3, I went ahead and visualized it. In the future if I find official country-wise rice cultivars, I shall update this chart, for now it simply serves as a learning purpose.



The final chart was this static chart below.

but I did post an interactive version of the chart as well on twitter, which is the gif below, merely to represent the sheer vastness of the indian rice varieties.

The static chart has a tooltip. Let's append a div element to the body and give it a class name called tooltip and hide it's visibility. Later using an event listener we will make the tooltip appear and disappear on user interaction.

//Create a div with hidden visibility which will pop up as tooltip 
//with rice cultivar names when user hovers over a grain.
const tooltip = d3.select("body")
    .append("div")
    .attr("class","tooltip")
    .style("visibility","hidden");

Since we're drawing the rice variety cultivars for each country, let's append a group element. Position the g element and create a placeholder for each country and bind the group data and join the g element. Now position these g elements using ternary operators to specify that if the index of the country is less than 17 space the elements 45 pixels from each other, if the index of the country is more than 17 space them 100 pixels from each other and reduce 900 pixels to position the country better. What does this mean? If you notice the chart, the first 17 countries have very few rice cultivars and take less space in terms of pixels. Therefore, Ive spaced them 45 pixels from each other. But countries after that starting with Nepal have a lot of rice varieties and require more space, so I've spaced them 100 pixels from each other and reduced 900 pixels to make it more evenly spaced.

//create a placeholder for each group country
const multiple = svg.append("g")
    .attr("transform",`translate(${width/2-120},10)`)
    .selectAll("country")
    .data(group)
    .join("g")
    .attr("transform",(d,i) => 
    `translate(0,${i<17?i*45:i*100-900})`)

Now lets draw the rice pictogram. Append a g element. Using d3.selectAll("use") create a placeholder for all the def elements and reuse the def. Bind the data and join the use element. Use href attribute to reference the rice path with id name "#rice". Set the stroke, stroke-width and other necessary attribute for the rice grain. Use the scale created earlier in transform, translate to position the rice grains.
Add event handlers to the rice elements (line 20-38) to make the tooltip appear on user interaction.




















 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 


//create a place holder for the rice grains inside each group country
const rice = multiple.append("g").attr("transform",`translate(0,0)`)

rice.append("g") //append group element
    .selectAll("use") //select use element to reuse the defs element
    .data(d => d[1]) //bind the data to the use element
    .join("use") //join the data
    .attr("href","#rice") //reference the rice grain path
    .style("stroke","black") //make the stroke of the rice grain black.
    .style("stroke-width",8) 
    .style("fill","#e0a871") //use a fill with desired colour.
    .attr("transform", (d,i) => 
    `translate(${scale(i%10*1.5)},${scale(Math.floor(i/10))}) 
    scale(0.12)`) //now place the grain using the scale we created earlier 
    //with each row having 10 grains
    //and size the grain at 0.12 , if you tweak this number, 
    //the grain will become big or small accordingly.

    //make the chart interactive to display tooltips on hover.
    .on("mousemove", function(event,d){
        d3.select(this)
          .style("opacity",1)
          .attr("stroke-width",3);

        tooltip.transition()
               .duration(200)
               .style("visibility","visible")

        tooltip.html(d.variety)
               .style("left",(event.x)+10+"px")
               .style("top",(event.y)+"px");
    })
    .on("mouseout", function(event,d){
        d3.select(this)
          .style("opacity",1)
          .attr("stroke-width",2.5);

        tooltip.transition().style("visibility","hidden");
    })
