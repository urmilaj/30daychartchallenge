# Create the chart with D3.js


  • Github link to chart (opens new window) and code repository. (opens new window)
  • Setup the basic html. Follow the instructions here.
  • Create the chart.

The first step is to load the data. Since the dataset is in csv format, using d3.csv we load it. (line-2)

The dataset contains two columns called year and area. Now we take this data and plot the year values in the x-axis and the area values on the y-axis.

To draw the line chart we set up the scale and axis.

Using d3.scaleTime and d3.axisBottom generate the x values and x-axis. scaleTime will parse through all the values in year column of the dataset using it's domain and convert it to pixel values of the svg using it's range. When we pass the scaleTime to d3.axisBottom it will draw out the x-axis. (line 7,17)

Using d3.scaleLinear and d3.axisLeft generate the y values and y-axis. Similiar to the x-axis, d3.scaleLinear will be used to convert area column in the dataset to pixel values and draw the axis using d3.axisLeft. (line 12,21)

Finally use d3.line to draw the actual line to complete the chart. (line 25)

Format year is used to convert the string year values on the dataset to machine readable time values. Its important to format any date values, such as year on this dataset using d3.scaleTime to render the chart without errors. (line-4)

//Load the csv data
d3.csv("data/forestArea.csv").then(function(data){
  //format year values
  const formatYear = d3.timeParse("%Y")

  //create x scale
  const x = d3.scaleTime()
    .domain(d3.extent(data,d=>formatYear(d.year)))
    .range([margin.left,width-margin.right])

  //create y scale
  const y = d3.scaleLinear()
    .domain(d3.extent(data,d=>d.area))
    .range([height-margin.bottom,margin.top])

  //create x axis
  const xAxis = g => g.attr("transform",`translate(0,${height-margin.bottom})`)
    .call(d3.axisBottom(x))

  //create y axis
  const yAxis = g => g.attr("transform",`translate(${margin.left},0)`)
    .call(d3.axisLeft(y))
  
  //usind d3.line method to draw the line chart
  const line = d3.line()
    .x(d=>x(formatYear(d.year)))
    .y(d=>y(d.area))
    .curve(d3.curveCatmullRom)

Since the scale, axis and line is all set up, its time to draw them all out on the svg.

  • Append a title for the chart.
  • Draw the x-axis.
  • Draw the y-axis.
  • Provide an annotation to the x-axis to indicate that the x-axis represents year.
  • Provide an annotation for the y-axis to indicate the y-axis represents forest area.
  • Finally draw the line to complete the line chart.
//title for the chart
svg.append("text")
  .attr("x",width/2)
  .attr("y",0)
  .style("text-anchor","middle")
  .style("font-size","3em")
  .style("fill","#6ef32b")
  .text("World Forest Area");

//draw x-axis
svg.append("g")
  .call(xAxis)
  .attr("color","#58b56b")
  .style("font-size","0.9em")
  .style("font-weight","bold");

//draw y-axis
svg.append("g")
  .call(yAxis)
  .attr("color","#58b56b")
  .style("font-size","0.8em")
  .style("font-weight","bold");

//annotate the x-axis
svg.append("text")
  .attr("x",width/2)
  .attr("y",height)
  .style("text-anchor","middle")
  .text("Year")
  .style("fill","#6ef32b")
  .style("font-weight","normal");

//annotate the y-axis
svg.append("text")
  .attr("x",-height/2+140)
  .attr("y",0)
  .text("Forest area (% of land area) in sq. km")
  .style("text-anchor","end")
  .attr("transform","rotate(-90)")
  .style("fill","#6ef32b");

//draw the line 
svg.append("path")
  .datum(data)
  .attr("fill","none")
  .attr("stroke","#6ef32b")
  .attr("stroke-width",3.5)
  .attr("d",line);
      
})
