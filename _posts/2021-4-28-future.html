  • Create a g variable and append a group element to the svg, set the transform translate attribute of this new g element. The transform and translate attribute positions all the circles, if these values are changed, it will move all the circles on the chart accodingly.
  • Give the circles a glow filter (line 4-14). Nadieh Bremer has an excellant blog post on this. (opens new window) Do check it out. Its very informative.
  • Now to insert the multiple globe images (line 16-23), create a placeholder for all the images using d3.selectAll, bind the dataset, join the images, using transform and translate attribute and the scales created earlier position all the images. Link the image using "xlink:href" attribute, use x, y to set the manipulate the image position and set desired opacity.
  • Now the code to position the circles are pretty much the same as the image, except since we're drawing circles, we create placeholders for circle, bind the data and join the circle. Similarly as for the image, position all the circles using transform and transalte. Set the radius, colour, fill attributes etc. (line 25-33)
  • Use the glow filter on the circles drawn with class name exo. (line35-36)
const g = svg.append("g")
    .attr("transform",`translate(${margin.left+30},${margin.top+80})`)

g.append("defs")
    .append("filter")
    .attr("id","glow")
    .append("feGaussianBlur")
    .attr("stdDeviation","3.5")
    .attr("result","coloredBlur")
    .append("feMerge")
    .append("feMergeNode")
    .attr("in","coloredBlur")
    .append("feMergeNode")
    .attr("in","SourceGraphic")

g.selectAll("image")
    .data(data)
    .join("image")
    .attr("transform",(d,i)=>`translate(${x(i%5)},${y(Math.floor(i/5))}) scale(0.077)`)
    .attr("xlink:href",img.src)
    .attr("x",-250)
    .attr("y",-250)
    .attr("opacity",0.3)

g.selectAll("circle")
    .data(data)
    .join("circle")
    .attr("class","exo")
    .attr("transform",(d,i)=>`translate(${x(i%5)},${y(Math.floor(i/5))})`)
    .attr("r",(d,i)=>d.Radius*radius)
    .attr("stroke",color)
    .attr("stroke-width",2)
    .attr("fill","none")

g.selectAll(".exo")
    .style("filter","url(#glow)")

g.selectAll("circles")
    .data(data)
    .join("circle")
    .attr("transform",(d,i)=>`translate(${x(i%5)},${y(Math.floor(i/5))})`)
    .attr("r",(d,i)=>d.Radius*radius)
    .attr("stroke",color)
    .attr("stroke-width",2)
    .attr("fill","none")

Notice each circle has text annotations attached to it, these annotations are made using d3-annotation library. Here's the code below to annotate the chart.

  • First map the data, set the x,y,dx,dy variables which will be used by the library to position the annotations.
  • Set the annotation colour.
  • The note will display the annotation texts.
  • Since the annotation I've chosen here is a 'd3.annotationCalloutCircle' shape, we can set a radius to it.
  • Set the type of your annotation.
  • Now use these variables inside d3.annotation().
  • Finally append a group element to the svg and call the annotation to draw it on the svg.
const annotations = data.map((d,i)=>{
return {
    x:x(i%5),
    y:y(Math.floor(i/5)),
    dx:0,
    dy:-60,
    color:color,
    note:{
    title: d.Name,
    label: d.Distance+" lightyears away."
    },
    subject:{
    radius:radius*d.Radius
    }
}
})

const type = d3.annotationCustomType(
    d3.annotationCalloutCircle,
    {"className":"custom",
    "note":{
    "align":"middle"}})

const makeAnnotations = d3.annotation()
    .type(type)
    .textWrap(110)
    .annotations(annotations)

svg.append("g")
    .attr("transform",`translate(${margin.left+30},${margin.top+80})`)
    .call(makeAnnotations)
