# Mount Everest Summits.

# Week 4: Timeseries. Day 20: Upwards.


For day 20's topic 'Upwards', I already had in mind that The Himalayan Database by Elizabeth Hawley existed and I was going to make a summits line chart out of that data but one extra factor I added was a cropped image of the Himalayas to give it an extra effect and this idea was definitely inspired by Mona Chalabi.

Over the years the number of people who made it to the summit of Mount Everest has increased and this fit the upwards topic perfectly and also the data itself represents individuals climbing to the top and its also a time series data, so it fit the week and day topic perfectly. I had a lot of fun making this.

  • Use d3.timeParse to use in the d3.scaleTime to make the year column data machine readable time format.
  • Use d3.scaleTime to help plot the x-axis and the line plot.
  • Use d3.scaleLinear to help plot the y-axis and the line plot.
  • Use d3.axisBottom to help draw the x-axis.
  • Use d3.axisLeft to help draw the y-axis.
  • Use d3.line to help draw the line plot.
const formatYear = d3.timeParse("%Y")

const x =  d3.scaleTime()
    .domain(d3.extent(data,d=>formatYear(d.year)))
    .range([margin.left+15,width-margin.right])

const y = d3.scaleLinear()
    .domain([5,d3.max(data,d=>d.count)])
    .range([height-margin.bottom,margin.top])

const xAxis = g => g.attr("transform",`translate(0,${height-margin.bottom})`)
    .call(d3.axisBottom(x))

const yAxis = g => g.attr("transform",`translate(${margin.left-10},0)`)
    .call(d3.axisLeft(y))
    .call(g => g.selectAll(".tick line")
            .clone()
            .style("stroke","#e0eddc")
            .attr("stroke-opacity",0.4)
            .attr("x2",width-margin.right-20))
    .call(g=>g.select(".domain").remove())

const line = d3.line()
    .x(d=>x(formatYear(d.year)))
    .y(d=>y(d.count))
    .curve(d3.curveCatmullRom)

  • To add the everest image on the chart, first append image on the svg. Use attribute 'href' which stands for hyperlink reference with value 'data/MountEverest.png' to pick out the png image file with the name 'MountEverest' from the data folder. Use 'x' and 'y' attribute to indicate where you would like to place that image on the svg. (line 2)
  • draw the x-axis and annotate the x-axis. (line 10 and 17)
  • draw the y-axis and annotate the y-axis. (line 27 and 34)
  • draw the line plot (line 44)
  • Insert the chart title. (line 52)
//append everest image
svg.append("image")
.attr("href","data/MountEverest.png")
.attr("width",width)
.attr("height",height-margin.bottom)
.attr("x",width/2-20)
.attr("y",5)

//draw the x-axis
svg.append("g")
.call(xAxis)
.attr("color","#e0eddc")
.style("font-weight","bold")
.style("font-size","1.1em")

//annotate the x-axis
svg.append("text")
.attr("x",width/2)
.attr("y",height)
.style("text-anchor","middle")
.text("Year")
.style("fill","white")
.style("font-size","1.5em")
.style("font-weight","normal");

//draw the y-axis
svg.append("g")
.call(yAxis)
.attr("color","#e0eddc")
.style("font-weight","bold")
.style("font-size","1.1em")

//annotate the y-axis.
svg.append("text")
.attr("x",-height/2+140)
.attr("y",-25)
.text("Number of summits")
.style("text-anchor","end")
.attr("transform","rotate(-90)")
.style("font-size","1.5em")
.style("fill","white");

//draw the line plot
svg.append("path")
.datum(data)
.attr("d",line)
.attr("fill","none")
.style("stroke","#e0eddc")
.style("stroke-width",3)

//chart title
svg.append("text")
.attr("x",width/2)
.attr("y",0)
.style("font-family","Bradley Hand ITC")
.text("Mount Everest Summits")
.style("fill","white")
.style("font-size","2.5em")
.style("text-anchor","middle")

Using Susie Lu's d3-annotation library annotate the chart with the number of people who made it to the summits over the years.

//map the data using scales to place the annotations
const annotations = data.map((d,i)=>{
return {
    x:x(formatYear(d.year)),
    y:y(d.count),
    dx:0,
    dy:-30,
    color:"white",
    className: "show-bg",
    note:{
    title: d.count + " successful summits"
    }
}
})

//use type to define the type of annotation to be used from the library.
const type = d3.annotationCustomType(
    d3.annotationLabel,
    {"className":"custom",
    "connector":{"end":"dot"},
    "note":{
    "align":"middle"}})

//use d3.annotation method to help draw the annotations
const makeAnnotations = d3.annotation()
    .type(type)
    .textWrap(10)
    .annotations(annotations)

//finally call make annotations variable to draw annotations on svg.
svg.append("g")
    .call(makeAnnotations)
