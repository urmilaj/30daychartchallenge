# Create the chart with D3.js and Zdog.


  • Github link to chart (opens new window) and code repository. (opens new window)

The above github repo link is only an iframe in the html body tag. I appended an iframe because this chart is a fork of Alok Pepakayala's notebook. (opens new window) Since I did not create the chart from scratch I've used an iframe.

So here's how I made this chart.


  • Step 1: Fork the notebook. Click the fork button. It will create a copy on your observable account.
  • Step 2: Click on the ellipsis and attach the csv data file.
  • Step 3: Find the chart_data cell on the notebook and load the csv data.
  • Step 4: Change the scale. Click on the scale cell and change the range value to 0,200.
  • Step 5: Change the svg height. Click on the illo cell and change the svg height to 800.
  • Step 6: Change background color. Click on the bg_color cell and change the hex value according to your liking.
  • Step 7: Change other attributes of the chart like size of the bars, font size, animation speed.
  • Step 8: Add annotations to the bar chart. If you notice the image below it has annotations on top of the bars to indicate the y-axis values. To add these annotations, click on the make_chart cell and set a new anchor called anchor3. Make changes to the text variable and add a new variable called text2 for the bar annotations.

That's it. Those are the changes required to re-create this chart.
