# Solar Maxima and Human Excitability

# Week 1: Comparisons. Day 4: Magical.


For day four I made a simple static line chart. The prompt for the day was magical, I did some googling and came across this page by Adam Michalec (opens new window) explaining historical events of human unrest or revolutions and comparing it to sun spot counts or solar maxima. It had data on solar cycle events and the historic events that took place. So I searched for data of sun spot counts and came across the National Centers for Environmental Information by NOAA, which had the sun spots data. I googled solar maxima data years for any historic events and annotated it on the chart. This correlation may not imply causation but it did sound magical and decided to use this data to make the chart for day four.

The final chart is below.

Load the data and set up the scales and axis for the chart.

// Use d3.timeParse("%Y") to convert the date format.
const formatYear = d3.timeParse("%Y");

// Load the csv data using promise.
d3.csv("data/solar.csv").then(function(data){

// create the x scale using d3.scaleTime()
const x = d3.scaleTime()
    .domain(d3.extent(data,d=>formatYear(+d.Year)))
    .range([margin.left,width-margin.right])

// create the y scale using d3.scaleLinear()
const y = d3.scaleLinear()
    .domain([0,d3.max(data,d=>+d.Wolf)])
    .range([height-margin.left,margin.top+100])

// create the x axis using the time scale.
const xAxis = g => g.attr("transform",`translate(0,${height-margin.bottom})`)
    .call(d3.axisBottom(x))

// create the y axis using linear scale.
const yAxis = g => g.attr("transform",`translate(${margin.left},0)`)
    .call(d3.axisLeft(y))

Use d3.line() to set up the pixel postions of the x and y points of the dataset. Append a group element to the svg and call the respective axis to draw the axis. Annotate each axis by append an individual text element to the svg.

// Create the line, using the x scales and y scales.
const line = d3.line()
    .x(d=>x(formatYear(+d.Year)))
    .y(d=>y(+d.Wolf))

// draw the x axis path.
svg.append("g")
    .call(xAxis)
    .call(g => g.select(".domain").remove())
    .style("color","#640e50")
    .style("font-size","1em")
    .style("font-weight","bold");

// Label the x axis
svg.append("text")
    .attr("x",width/2)
    .attr("y",height+10)
    .text("Year")
    .style("fill","#640e50")
    .style("font-size","1.5em")
    .style("font-weight","bold")
    .style("text-anchor","middle");

// draw the y axis path.
svg.append("g")
    .call(yAxis)
    .call(g => g.select(".domain").remove())
    .style("color","#640e50")
    .style("font-size","1em")
    .style("font-weight","bold")
    .style("display","none");

// label the y axis
svg.append("text")
    .attr("transform","rotate(-90)")
    .attr("x",-height/2-50)
    .attr("y",-10)
    .text("Wolf Number")
    .style("fill","#640e50")
    .style("font-size","1.5em")
    .style("font-weight","bold")
    .style("text-anchor","middle");

Draw the line plot by appending a path to the svg, bind the datum and provide attributes to the path. Set the "d" attribute which will draw the line plot.

// draw the path since it's a single line use datum to bind the data
// draw it using the attribute "d" 
svg.append("path")
    .datum(data)
    .attr("fill","none")
    .attr("stroke","#e99a66")
    .attr("stroke-width", 2.5)
    .attr("stroke-linejoin", "round")
    .attr("stroke-linecap", "round")
    .attr("d",line)
