# Sacred Groves of India.

# Week 3: Relationships. Day 16: Trees.


Day 16's topic was trees. So I visualized India's sacred groves. Sacred groves are forest areas which are dedicated to specific deities, this helps in forest conservation and biodiversity. The final visualization below is a circular bubble plot, each bubble representing each Indian state and the size of the bubble shows the number of sacred groves in each state.

Set up the scales for the circular bubble plot. Using d3.scaleBand we can place the circles at an equal distance in a circle path. Using d3.scaleRadial we can create the circular path. d3.scaleSqrt will help size are circles between the range of 5 pixels radius and 60 pixels radius. Therefore states that have the highest sacred groves will have a big bubble with a radius of 60 pixels and smaller bubbles with radius of 5 pixels will represent states with lower number of sacred groves.

const x = d3.scaleBand()
    .domain(data.map(d=>d.state))
    .range([0,2*Math.PI])

const y = d3.scaleRadial()
    .domain([0,d3.max(data,d=>d.documented_sacred_forests)])
    .range([innerRadius,outerRadius])

const radius = d3.scaleSqrt()
    .domain(d3.extent(data,d=>d.documented_sacred_forests))
    .range([5,60])

  • Append a group element and use selectAll to create a placeholder for the circles, bind the data and use join to create the circle elements.
  • Using transform, translate attribute and the scales created previously position the circles.
  • Use attribute r and radius scale to size the circles as per the data.
  • Colour the circles and the chart is ready.
g.append("g")
    .selectAll("circle")
    .data(data)
    .join("circle")
    .attr("transform", (d,i) => `
                    rotate(${((x(d.state) + x.bandwidth() / 2) * 180 / Math.PI - 90)})
                    translate(${position},0)
                `)
    .attr("r",d=>radius(d.documented_sacred_forests))
    .attr("fill","#60b26f")
    .attr("stroke","white")
    .attr("stroke-width",2.5)
    .style("opacity",0.8)

Finally add some annotations for the circles and use the code below to position the annotations circularly.

g.append("g")
    .selectAll("g")
    .data(data)
    .enter()
    .append("g")
    .attr("text-anchor", function(d) { return (x(d.state) + x.bandwidth() / 2 + Math.PI) % (2 * Math.PI) < Math.PI ? "end" : "start"; })
    .attr("transform", function(d) { return "rotate(" + ((x(d.state) + x.bandwidth() / 2) * 180 / Math.PI - 90) + ")"+"translate(" + (y(d.documented_sacred_forests)+10) + ",0)"; })
    .append("text")
        .text(function(d){return(d.state+ " : " + d.documented_sacred_forests)})
        .attr("transform", function(d) { return (x(d.state) + x.bandwidth() / 2 + Math.PI) % (2 * Math.PI) < Math.PI ? "rotate(180)" : "rotate(0)"; })
        .attr("alignment-baseline", "middle")
