# Week 3: Relationships. Day 15: Multivariate.


Day 15's topic was multivariate. This dataset (opens new window) on UCI machine learning repository contains multivariate data on 100 volunteers, whose semen samples were analysed to determine their feritlity. There were a total of eight categories, with respective attributes. I decided to visualize this data in the form of a parallel coordinates chart.

Here's the final chart below.

  • Each volunteer was between the age group of 18 - 36 years of age.
  • Volunteers who had suffered any childhood diseases were marked 0, if yes. 1 if they had no childhood diseases.
  • Similarly if a volunteer had suffered accident or serious trauma were marked 0, if yes. If they had no such accident/trauma they were marked 1.
  • They same was analysed for any surgical intervention in volunteers. 0: yes, 1: no.
  • Volunteers were asked about any high fevers they had in the last year. If a volunteer had a fever less than 3 months ago: -1, more than 3 months ago: 0, no: 1.
  • Volunteers were analysed for alcohol consumption:- several times a day: 0.2, everyday: 0.4, several times a week: 0.6, once a week: 0.8, hardly or never: 1
  • Volunteers were also analysed for smoking habits:- never: -1, occasional: 0, daily: 1
  • Volunteers were also asked about the number of hours spent sitting in a day:- 0,1

Based on this analysis, volunteers results are marked in white on the chart if their diagnosis was normal. If volunteer results were altered, these results were marked in blue.

# Create the chart with D3.js

Load the csv data using promise.

d3.csv("data/fertilityData.csv").then(function(data){})

Extract the names of the columns from the dataset, we only need the first eight columns , therefore use code below to do this. These columns will be used as the axis for the parallel coordinates chart.

const keys = data.columns.slice(0,8);
1

Set up the scales for the chart. Use d3.scalePoint for the x-axis, this will draw the line from point to point for each volunteer based on the categorical attributes. Y-axis is drawn using d3.scaleLinear() since each category is analysed on a scale of 0 to 1. But each y-axis varies from the other, therefore create a map for each category from the dataset.

const x = d3.scalePoint(keys, [margin.left, width - margin.right]);
const y = new Map(Array.from(keys, key => [key, d3.scaleLinear(d3.extent(data, d=> d[key]), [margin.top, height - margin.bottom])]));
1
2

Use d3.line, which will help us draw the line on the parallel coordinates chart for each volunteer based on the x and y values.

const line = d3.line()
    .defined(([, value]) => value != null)
    .x(([key]) => x(key))
    .y(([key, value]) => y.get(key)(value));

Set the colour for the chart line using d3.scaleOrdinal. The range indicates if the diagnosis is normal that is "N" colour the line white, otherwise colour the line blue ( hex vale 1f15e6), if the diagnosis is altered indicated by "O" on the dataset.

const color = d3.scaleOrdinal()
    .domain(data.map(d=>d.Diagnosis))
    .range(["white","#1f15e6"]);

//chart title
svg.append("text")
.attr("x",width/2)
.attr("y",-50)
.style("font-weight","bold")
.style("font-size","1.7em")
.style("text-anchor","middle")
.text("Fertility")

//draw the line path
svg.append("g")
.attr("fill","none")
.attr("stroke-width",1.5)
.attr("stroke-opacity", 1)
.selectAll("path")
.data(data)
.join("path")
.attr("stroke",d=>color(d.Diagnosis))
.attr("d", d=>line(d3.cross(keys, [d], (key, d) => [key, d[key]])))

//draw the y-axis
svg.append("g")
.selectAll("g")
.data(keys)
.join("g")
.attr("transform", d=> `translate(${x(d)})`)
.each(function(d) {d3.select(this).call(d3.axisLeft(y.get(d)))})
.style("stroke-width",2)
.style("font-weight","bold")

Finally annotate the chart to indicate axis categories using d3.annotation.

const annotations = keys.map((d,i)=>{
return{
x:x(d),
y:95,
dy:-30,
dx: 0,
color:"black",
note:{
    title:d
}
}})

const type = d3.annotationCustomType(
d3.annotationLabel,
{"className":"custom",
"connector":{"end":"dot"},
"note":{"lineType":"horizontal",
"align":"middle"}})

const makeAnnotations = d3.annotation()
.type(type)
.textWrap(80)
.annotations(annotations)

svg.append("g")
.call(makeAnnotations)
