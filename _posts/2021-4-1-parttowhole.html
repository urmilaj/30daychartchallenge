Once the data is loaded, lets now add a chart title. (line 5-10) To add a chart title, lets append a 'text' element to the svg, set the x and y attributes of this text element, which will position the chart title. Attribute x has been set to width/2, width here is the svg's width which is 960 pixels, so the svg starts at 0 pixels and extends to 960 pixels, when we specify width/2, it is 960/2, i.e 480 pixels, which is the middle of the width. Attribute y is set at -150. This is because of margin spacing which has been set to 200. Finally set the font-size and text-anchor styling. The title of the chart is 'Phylogeny'. Set the chart title using the text method.



 
 




// insert chart title.
svg.append("text") // indicates a text element placeholder.
   .attr("x", width/2) // use x,y to place the text
   .attr("y",-150)
   .style("font-size","2.5em") 
   .style("text-anchor","middle")
   .text("Phylogeny") // the actual text itself.

To create the dendogram, the data needs to be in a specific format. Using d3.hierarchy will help us in creating the required format which is a nested object. Tweaking the root.dx and root.dy values will change the shape of the dendogram making it narrow or wide in shape. Play around with these values, to see how it works.

// d3.hierarchy puts data in a hierarchial format from root node.
const root = d3.hierarchy(data)
  .sort((a,b) => (a.height-b.height))

root.dx = -5; //tweak dx, dy value to make the dendogram wide or narrow
root.dy = 100;

To draw the dendogram. Append a group element to the svg. Create a placeholder for all the paths using d3.selectAll('path'), bind the hierarchial data and join the path. line (6-8). Add stroke, stroke-width, fill attributes to the dendogram. The actual path itself is drawn using the d attribute. (line 10-14)



To animate the dendogram use stroke-dasharray to get the total length of the dendogram path, using transition and delay methods in d3 to animate it. (line 15-23) Transition will add an appearing effect. If just use transition all the dendogram paths will appear at once. Adding delay and specifying i*500, will make each index path to appear delayed or one after the other.






 
 
 

 
 
 
 
 










const link = svg.append("g") //append a group element and position it 
  .attr("transform",`translate(${width/2-300},${height/2+120})`)
  .attr("fill","transparent") //transparent fill
  .attr("stroke","purple") //colour of the dendogram's stroke
  .attr("stroke-width",2.5) //desired stroke-width
  .selectAll("path") //create a placeholder for the path
  .data(root.links()) //bind the data
  .join("path")
  .attr("class","path") //give it a class name
  .attr("d", d=>        //draw the path from data
    `M${d.target.y},${d.target.x}
    C${d.source.y + root.dy / 2},${d.target.x}
    ${d.source.y + root.dy / 2},${d.source.x}
    ${d.source.y},${d.source.x}`)
  .attr("stroke-dasharray",function() //creates the transition effect of path
    {return pathLength = this.getTotalLength()})
  .transition()
  .delay((d,i)=>i*500)
  .attr("stroke-dasharray",pathLength)
  .transition()
  .attr("stroke-dasharray",0)
  .transition()
  .attr("stroke-dasharray",0)

Annotate the dendogram. Append a group element, set the position of the g element using transform, translate attribute. Append a new group element, bind the hierarchial data to it and join it, position this g element using hierarchial position of x and y. Now append a text element and position the text attribute using x. Using ternary operator specify is the text element is children place it 6 pixels and if its a parent place it at -6 pixels.

The annotation animates along with the denodgram. To create this animation use transition, duration and delay method to change the opacity of the text from 0 to 1. (line 14-17)

const label = svg.append("g") //append group element for naming the species.
  .attr("transform",`translate(${width/2-300},${height/2+120})`)
  .selectAll("g") //select all the group element
  .data(root.descendants()) //bind the group to the data
  .join("g") //join or create the group elements
  .attr("transform", d=> `translate(${d.y},${d.x})`) //position the group

label.append("text") //append text
  .attr("x", d=>d.children?-6:6) //if text is child node position at 6 else -6
  .text(d=>d.data.name) //the name of the species
  .style("font-size","1.3em") //font size, text position and opacity set to 0
  .attr("text-anchor", (d,i)=>d.children?"end":"start")
  .style("opacity",0)
  .transition() //create a transition effect for text to appear
  .duration(2000) 
  .delay((d,i)=>i*400)
  .style("opacity",1) //set opacity to 1 to display text.
