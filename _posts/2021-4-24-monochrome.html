# Create the chart with D3.js


  • Github link to chart (opens new window) and code repository. (opens new window)
  • Setup the basic html. Follow the instructions here.
  • Create the chart.

Load the json data.

d3.json("data/ghibli.json").then(function(data){})
1

Set up the scales required to create the chart. The radials are created using d3.arc. The x and y scales are used to place the five radials horizontally and vertically. This is done by specifying it in the domain of the x and y scales. Create a score scale which will scale the radials score for each film.

const films = data.films

const x = d3.scaleLinear()
    .domain([0,5])
    .range([margin.left,width])

const y = d3.scaleLinear()
    .domain([0,5])
    .range([margin.top,height-margin.bottom])

const score = d3.scaleLinear()
    .domain([0,d3.max(films,d=>+d.rt_score)])
    .range([0,Math.PI*2])

const arc = d3.arc()
    .startAngle(0)
    .endAngle(d=>score(+d.rt_score))
    .innerRadius(40)
    .outerRadius(50)

To create the radials, appens a group element, use transform, translate to specify the position of the group elements.
Select all the path, bind the data and join the paths. Use the x scale and y scale created earlier inside transform, translate attribute to position each radial.
Now that the positions are set, use the d attribute to draw the arc paths and add a fill to the arc.
Each arc has a specific score, Append a group element and add text to specify each movie's score in the middle of the radial. 

svg.append("g")
.attr("transform",`translate(${margin.radialLeft},${margin.radialTop})`)
.selectAll("path")
.data(films)
.join("path")
.attr("transform",(d,i)=>`translate(${x(i%5)},${y(Math.floor(i/5))})`)
.attr("d",arc)
.attr("fill","white")

svg.append("g")
.attr("transform",`translate(${margin.radialLeft},${margin.radialTop})`)
.selectAll("text")
.data(films)
.join("text")
.attr("transform",(d,i)=>`translate(${x(i%5)},${y(Math.floor(i/5))})`)
.text(d=>d.rt_score)
.style("fill","white")
.style("text-anchor","middle")

Each radial has annotations specifying the name of the movie it represents. Add these annotations using Susie Lu's d3-annotation library.

const annotations = films.map((d,i)=>{
return {
    x:x(i%5),
    y:y(Math.floor(i/5)),
    dx:0,
    dy:-48.5,
    color:"black",
    note:{
    title: d.title+ ", "+d.release_date,
    }
}
})

const type = d3.annotationCustomType(
d3.annotationLabel,
{"className":"custom",
"note":{
"align":"middle"}})

const makeAnnotations = d3.annotation()
    .type(type)
    .textWrap(120)
    .annotations(annotations)

svg.append("g")
    .attr("transform",`translate(${margin.radialLeft},${margin.radialTop})`)
    .call(makeAnnotations)
