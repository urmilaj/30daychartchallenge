# On This Day

# Week 1: Comparisons. Day 3: Historical.


Day three's prompt was historical. I was feeling super creative and had this idea to use qualitative data from On this day (opens new window) , this website has historical information of events that occurred over the years on a particular day. The data had the following main categories: 'History', 'Film and Tv', 'Music' and 'Sport'. Each category had sub categories which were important events, birthdays and deaths. I decided to visualize the data in the form of a photo frame with tidy trees popping out from each side of the frame to represent the main category and colours to represent the sub categories and made it interactive, when the user hovers over the end node circles the event would pop up in the middle of the frame.

Create a tooltip div element. First select the body html element and append a div to it, give it a class and then append a paragraph element to contain the main text on the center of the frame. Providing a class to your tooltip will help later to style it using css.

//Select the body of the html and append a div for tooltip
const tooltip = d3.select("body")
    .append("div")
    .attr("class","tooltip")
    .append("p")
    .attr("class","chartTitle") 
    .text("ON THIS DAY, APRIL 3")

Now create four group circles and position them around the frame which will be the position for the final tidy trees. It would look like this.

Add the circles which have a radius of 10 and cx and cy attributes help place the circles around the mid-points of each side of the rectangle. Each circle is a data point with index values. Use javascript ternary operators to evaluate that if the index of the data point is even use the x scale created earlier to place cx at that scale point otherwise place it at 60 pixels. Use the same evaluation for cy. You can play around with this to place it any other point, if you'd like.

//append group element to svg, selectAll the circles and bind to the data.
//Use d3.join() to join the circles and all the tidy trees to it.
const day = svg.append("g")
    .attr("transform",`translate(${width/2-60},${height/2-120})`)
    .selectAll("circle")
    .data(group)
    .join(enter => {
        const circle = enter.append("circle")
            .attr("r",10)
            .attr("cx", (d,i) => i%2==0 ? x(i):60)
            .attr("cy", (d,i) => i%2==0 ? 120:y(i))
            .attr("fill","black")
            .attr("stroke","black")
            .style("stroke-width",2)
            .style("display","none")

Now select the circles that we have created and make a tidy tree for each of them. Set up the data in a suitable format using d3.hierarchy(), d3.tree() and d3.linkRadial() which will help to draw the final tidy trees.

//selectall the circle and use each function to create 
//the individual tidy trees for each main category
svg.selectAll("circle")
    .each((d,i) => {

//Use d3.hierarchy() to create a hierarchial format of the data
    const hierarchy = d3.hierarchy(d[1])

//d3.tree() creates the tidy trees, in 180 degree format.
    const tree = d3.tree()
        .size([Math.PI, 180])

//Pass the hierachial data format to the tree which help draw it.
    const root = tree(hierarchy)

//Create the links in a circular or radial format using d3.linkRadial()
    const linkRadial = d3.linkRadial()
        .angle(d => d.x)
        .radius(d => d.y)

Create a group element and use transform, translate to change the direction each tidy tree faces. I wanted each tidy tree to face outward from the frame. To do this use the rotate attribute and index of the data point to turn it in your desired manner. I use ternary operators to rotate it, based on the index.

//Draw the trees.        
svg.append("g")
    .attr("transform",`translate(${width/2-60},${height/2-120})
        translate(${i%2==0 ? x(i):60}, ${i%2==0 ? 120:y(i)})
        rotate(${i==0 ? -180:0})
        rotate(${i==1 ? -90:0})
        rotate(${i==3 ? 90:0})`)
    .selectAll("radial")
    .data(root.links()) //use root.links() to bind the data.
    .join("path")
    .attr("class","day")
// the d attribute draws the path of the tidy tree and it's branches.
    .attr("d", i%2==0 ? linkRadial.radius(d => d.y*1.3):linkRadial.radius(d => d.y*1.35))
    .style("fill","none")
    .style("stroke", d=> d.source.depth==1 ? color(d.source.data[0]):"#86569a")
    .style("stroke-width",1.9)

Create the end node circles which is interactive. When the user hovers over these node circles the tooltip will pop up in the middle of the frame. Use root.descendants() and filter the data to depth equals two, which will places these circles only at the edge of the branches of the tidy trees. Otherwise the circles will be appended at all points of the root like this.
If you would like to keep these circles, do not filter, but since these circles are interactive and make tooltips pop up, when user hovers over these extra circles, there would be errors in the tooltip. Therefore I have filtered it and here's the code for the circles.

svg.append("g")
    .attr("transform",`translate(${width/2-60},${height/2-120})
        translate(${i%2==0 ? x(i):60}, ${i%2==0 ? 120:y(i)})
        rotate(${i==0 ? -180:0})
        rotate(${i==1 ? -90:0})
        rotate(${i==3 ? 90:0})`)
    .selectAll("circles")
    .data(root.descendants().filter(d => d.depth==2))
    .join("circle")
    .attr("transform", d => `rotate(${i%2 == 0?d.x*180 / Math.PI - 90:d.x*180/Math.PI-90})
        translate(${i%2==0?d.y*1.32:d.y*1.37},0)`)
    .style("fill", d => color(d.data.type))
    .style("stroke",d => color(d.data.type))
    .attr("r",6)
    .on("mouseover touchstart", function(event, d){
        d3.select(this)
            .style("fill","white")
            .attr("r",15)
        
        tooltip.html("This day in "+d.data.category+"<img src='"+d.data.image+"' width='250' height='280'></img>"+d.data.details)
            .style("font-size","1.2em")
        })
    .on("mouseout touchend", function(event, d){
        d3.select(this)
            .style("fill",d => color(d.data.type))
            .attr("r",6)
        
        tooltip.html("ON THIS DAY, APRIL 3")
            .style("font-size","4.5em")
        })
