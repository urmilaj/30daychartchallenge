Set up the scales, axis and line necessary to draw the line chart.

//format year column using timeParse.
const formatYear = d3.timeParse("%Y");

//set up x-scale.
const x = d3.scaleTime()
.domain(d3.extent(data,d=>formatYear(d.year)))
.range([margin.left,width-margin.right]);

//set up x-axis
const xAxis = g => g.attr("transform",`translate(0,${height-margin.bottom})`)
.call(d3.axisBottom(x));

//set up y-scale
const y = d3.scaleLinear()
.domain(d3.extent(data,d=>d.crude_birth_rate))
.range([height-margin.bottom,margin.top]);

//set up y-axis
const yAxis = g => g.attr("transform", `translate(${margin.left},0)`)
.call(d3.axisLeft(y));

//d3.line to draw the line plot
const line = d3.line()
.x(d=>x(formatYear(d.year)))
.y(d=>y(d.crude_birth_rate))
.curve(d3.curveMonotoneX);

To make the line chart transition first append the path.
Get the total length of the path.
Use stroke dasharray and stroke dashoffset to make the line draw from the start to the end.
Use d3.transition() to animate the line and d3.easeLinear will add a smooth effect when the line tranisitions. Specify duration to determine how fast the line animates, the lower the number the faster the animation and a higher number will make the line animate/transition slower.

//append the path and bind the data
//use attribute d to draw the line
const path = svg.append("path")
.datum(data)
.attr("d",line)
.attr("fill","none")
.attr("stroke","white")
.attr("stroke-width",3.5)

//get the total length of the line path
const length = path.node().getTotalLength()

//animate the path
path.attr("stroke-dasharray",length+" "+length)
.attr("stroke-dashoffset",length)
.transition()
.ease(d3.easeLinear)
.attr("stroke-dashoffset",0)
.duration(5500)

Finally draw the x-axis and y-axis and annotate the axis to complete the chart.

//draw x-axis
svg.append("g")
.call(xAxis)
.attr("color","#652bc9")
.style("font-weight","bold")

//draw y-axis
svg.append("g")
.call(yAxis)
.attr("color","#652bc9")
.style("font-weight","bold")

//annotate the x-axis
svg.append("text")
.attr("x",width/2)
.attr("y",height)
.style("text-anchor","middle")
.style("fill","#652bc9")
.style("font-size","1.8em")
.text("Year")

//annotate the y-axis
svg.append("text")
.attr("x",-height/2+190)
.attr("y",10)
.text("Crude Bith Rate (per 1000 people)")
.style("text-anchor","end")
.attr("transform","rotate(-90)")
.style("font-size","1.8em")
.style("fill","#652bc9");
