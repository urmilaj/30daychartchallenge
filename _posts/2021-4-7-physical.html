# Week 2: Distributions. Day 7: Physical.


Honestly day seven was a fail for me. I did not know what my chart would be about or what data I could use. Then I came across this observable notebook by Mike Bostock. (opens new window) The notebook makes an image histogram, submit a picture to the notebook and it will make a histogram of colour distribution of red, green and blue values. So I used this notebook to create an image histogram of Raja Ravi Varma's paintings.

Here's an example below. This is red, green and blue colour distribution for one of Raja Ravi Varma's paintings called "Woman holding a fruit."


Since I forked a notebook and made a chart for day seven I have no code explanations. But I have it on my github repository using Observable's embed notebook features, that makes the chart available on GitHub pages.


Embed the notebook in vanilla html like so. Use notebook actions on Observable and click on embed cells.


Use the entire notebook option and copy it and paste the iframe html tag anywhere on your html file and the notebook contents will be embeded. Tweak the height and width attributes to embed in your desired way.
