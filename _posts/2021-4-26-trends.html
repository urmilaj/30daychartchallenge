  • Group the csv data based on income group. (line 1)
  • Parse the year data with d3.timeParse to convert it to datetime format. (line 3)
  • Create x scale. The x-scale in years on our chart. Use d3.scaleTime to create the x-scale which will used later to draw x-axis and the line on the chart. (line 5)
  • Create y scale. The y-scale us a linear scale of numbers, using d3.scaleLinear create the y scale, this will be used to draw the y-axis and line on the chart. (line 12)
  • Create the x-axis using d3.axisBottom. (line 9-10)
  • Create the y-axis using d3.axisLeft. (line 16-17)
  • Now each svg chart has two lines on the chart, one line represents the male demographic and the other line represents the female demographic. (line 19-26)
  • The dataset also contains information on high and low values for each demographic, this information is shaded on the line using d3.area. (line 27-35)
  • The colour to differentiate the male and female demographic lines is done using d3.scaleOrdinal. (line 37-38)
const group = d3.group(data, d => d.Income_Group)

const formatYear = d3.timeParse("%Y")

const x = d3.scaleTime()
    .domain(d3.extent(data, d => formatYear(d.Year)))
    .range([margin.left,width-margin.right])

const xAxis = g => g.attr("transform",`translate(0,${height-margin.bottom})`)
    .call(d3.axisBottom(x))

const y = d3.scaleLinear()
    .domain([0,50])
    .range([height-margin.bottom,margin.top])

const yAxis = g => g.attr("transform", `translate(${margin.left},0)`)
    .call(d3.axisLeft(y))

const maleLine = d3.line()
    .x(d => x(formatYear(d.Year)))
    .y(d => y(d.maleMedian))

const femaleLine = d3.line()
    .x(d => x(formatYear(d.Year)))
    .y(d => y(d.femaleMedian))

const maleArea = d3.area()
    .x(d => x(formatYear(d.Year)))
    .y0(d => y(d.maleHigh))
    .y1(d => y(d.maleLow))

const femaleArea = d3.area()
    .x(d => x(formatYear(d.Year)))
    .y0(d => y(d.femaleHigh))
    .y1(d => y(d.femaleLow))

const color = d3.scaleOrdinal()
    .range(["#e3e077","#f19087"])

The chart are made using a reusable chart function. Lets create the chart function.

  • Each time the chart function is called a new svg gets appended. Therefore let create an svg inside the function. Select the div with the class name "chart" and append an svg with the width, height and styling attributes.
  • Create the chart title, the chart's title will change according to the group data.
  • Annotate the x-axis and y-axis. (line 23-39)
  • draw the x-axis and y-axis. (line 41-49)
  • shade the line of the male and female demographics to indicate the high and low values. (line 52-57, line 68-73)
  • draw the line for the male and female demographic. (line 60-65, line 76-81)
  • insert the legend (line 84-105)
chart = (data, title) => {

//create the svg
    const svg = d3.select(".chart")
        .append("svg")
        .attr("width",width+margin.left+margin.right)
        .attr("height",height+margin.top+margin.bottom)
        .style("background-color","#7b81ad")
        .append("g")
        .attr("transform",`translate(${margin.left},${margin.top})`)

//insert chart title
    svg.append("text")
        .attr("x",width/2)
        .attr("y",40)
        .text(title)
        .style("text-anchor","middle")
        .style("font-size","1.5em")
        .style("fill","white")
        .style("font-family","Times New Roman")
        .style("font-weight","bold")

//annotate the x-axis
    svg.append("text")
        .attr("x",width/2)
        .attr("y",height-10)
        .style("text-anchor","middle")
        .style("fill","white")
        .style("font-family","Times New Roman")
        .text("Year")

//annotate the y-axis
    svg.append("text")
        .attr("x",-height/2-120)
        .attr("y",10)
        .text("Estimated death rate per 100,000 pop.")
        .attr("transform","rotate(-90)")
        .style("font-family","Times New Roman")
        .style("fill","white");

//draw the x-axis
    svg.append("g")
        .call(xAxis)
        .attr("color","white")

//draw the y-axis
    svg.append("g")
        .call(yAxis)
        .attr("color","white")

//shade the line for the male demographic
    svg.append("path")
        .datum(data)
        .attr("d",maleArea)
        .attr("fill","#e3e077")
        .attr("stroke","none")
        .style("opacity",0.4)

//draw the line for the male demographic
    svg.append("path")
        .datum(data)
        .attr("d",maleLine)
        .attr("fill","none")
        .attr("stroke","#e3e077")
        .attr("stroke-width",1.5)

//shade the line for the female demographic
    svg.append("path")
        .datum(data)
        .attr("d",femaleArea)
        .attr("fill","#f19087")
        .attr("stroke","none")
        .style("opacity",0.4)

//draw the line for the female demographic
    svg.append("path")
        .datum(data)
        .attr("d",femaleLine)
        .attr("fill","none")
        .attr("stroke","#f19087")
        .attr("stroke-width",1.5)

//draw the legend for each svg
    const legend = svg.append("g")
        .attr("transform",`translate(${width-margin.right-10},20)`)

    legend
        .selectAll("rect")
        .data(["Male","Female"])
        .join("rect")
        .attr("x",-35)
        .attr("y",(d,i)=>i*15)
        .attr("width",10)
        .attr("height",10)
        .attr("fill",d=>color(d))
    
    legend
        .selectAll("text")
        .data(["Male","Female"])
        .join("text")
        .text(d=>d)
        .attr("x",-20)
        .attr("y",(d,i)=>i*15+10)
        .style("fill","white")
        .style("font-size","0.85em")
}
