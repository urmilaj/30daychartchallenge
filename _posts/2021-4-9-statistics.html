# Bob Ross Paintings.

# Week 2: Distributions. Day 9: Statistics.


'Statistics' was the prompt for day nine. I visualized Bob Ross's paintings in the form of a bee swarm plot showing the number of colours he used for each painting. As shown in the chart below, most of them, a 100 to be precise used twelve colours and the rest of his paintings varied between one colour to a maximum of fifteen colours.

The final chart is below.

Set up the padding, radius, scales and axis for the chart.

const padding = 2;
const radius = 3.6;

const y = d3.scaleLinear()
    .domain(d3.extent(data,d=>+d.num_colors))
    .range([height,margin.top]);

const yAxis = g => g.attr("transform",`translate(${margin.left},0)`)
    .call(d3.axisLeft(y))
    .call(g => g.selectAll(".tick line")
    .clone()
    .style("stroke","#e0eddc")
    .attr("stroke-opacity",0.4)
    .attr("x2", width-margin.right))
    .call(g => g.select(".domain").remove())

svg.append("g")
    .call(yAxis)
    .style("color","white")

Use d3's forcesimultaion to position our circles on the chart.

const simulation = d3.forceSimulation(data)
    .force("x",d3.forceX(width).strength(0.002))
    .force("y",d3.forceY(d=>y(+d.num_colors)).strength(1))
    .force("charge",d3.forceManyBody().strength(0.01))
    .force("collide",d3.forceCollide(radius+padding).strength(2))
    .alpha(5)
              
for (let i = 0; i < 250; ++i) {simulation.tick()}

Finally let's append the circles in a beeswarm plot. The chart is interactive. If the user clicks on any of the circles, they'll be redirected to a youtube tutorial of how to make the painting by Bob Ross.
The highlighted lines below which is 4,5,6 allows us to redirect to a youtube page. Append "a" which is a html hyperlink tag, add the attribute and refer to the data in the column name youtube_src, the attribute target blank allows us to open the youtube link in a new tab.


As the user hover over the circle to click there will also be tooltip that pops up for each circle with the name of the painting the circle represents as highlighted in the code on line 14 and 26. Use a .on() event handler to make the tooltip appear and disappear on mousemove and mouseout resepectively.




 
 
 







 











 




svg.selectAll("circle")
    .data(simulation.nodes())
    .enter()
    .append("a")
    .attr("href",d=>d.youtube_src)
    .attr("target","_blank")
    .append("circle")
    .attr("cx",d=>d.x)
    .attr("cy",d=>d.y)
    .attr("fill",d => d.Titanium_White==0?"#da7b16":"transparent")
    .attr("stroke",d => d.Titanium_White==0?"#da7b16":"white")
    .attr("stroke-width",2)
    .attr("r",radius)
    .on("mousemove",function (event,d){
        d3.select(this).style("opacity",1).attr("stroke-width",3)
                      
        tooltip.transition()
            .duration(200)
            .style("visibility","visible")
              
        tooltip.html(`<h2>${d.painting_title}</h2><br/>`)
            .style("transform","translateY(-55%)")  
            .style("left",(event.x)+5+"px")
            .style("top",(event.y)+"px")
        })
    .on("mouseout",function (event,d){
        d3.select(this).style("opacity",1).attr("stroke-width",2)
        tooltip.transition().style("visibility","hidden")
        })

Finally annotate the chart using Susie Lu's d3.annotation.

const annotate = [{
    type : d3.annotationCustomType(
        d3.annotationCalloutElbow, 
        {"className":"custom",
        "connector":{"type":"elbow",
        "end":"dot"},
        "note":{"lineType":"horizontal",
        "align":"left"}}),
    note: {
        label: "Out of 403 paintings created, 100 of Bob Ross's paintings used 12 colours.",
        wrap:160,
          },
        x:630,
        y:230,
        dx:15,
        dy:-40,
        color:"white"
      },
      {
    type : d3.annotationCustomType(
        d3.annotationCalloutElbow, 
        {"className":"custom",
        "connector":{"type":"elbow",
        "end":"dot"},
        "note":{"lineType":"horizontal",
        "align":"left"}}),
    note: {
        label: "This painting called 'Contemplative Lady', used only one colour. The colour used was Van Dyke Brown or #221B15 in hex value.",
        wrap:160,
          },
        x:325,
        y:900,
        dx:155,
        dy:-40,
        color:"white"
      },
      {
    type : d3.annotationCustomType(
        d3.annotationCalloutElbow, 
        {"className":"custom",
        "connector":{"type":"elbow",
        "end":"arrow"},
        "note":{"lineType":"horizontal",
        "align":"right"}}),
    note: {
        label: "This painting called 'Pot O' Posies', along with the other circles which are orange, were the only paintings that did not use the colour 'Titanium White'.",
        wrap:160,
          },
        x:270,
        y:590,
        dx:-55,
        dy:-40,
        color:"#da7b16"
      }
    ].map(d => {return d})

    const makeAnnot = d3.annotation()
        .type(d3.annotationLabel)
        .annotations(annotate)

    svg.append("g")
        .call(makeAnnot)
    })
