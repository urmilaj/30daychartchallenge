Create the functions required to make reusable a reusable chart which will be used with the grouped csv data.
To do this first group the csv data using d3.group. (line 1)


Create the x and y scales.

  • The x-scale scales time, in this chart it scales the year 2000 to 2018 and turns into pixel spacing based on svg width specifications.
  • The y-scale scales linear values. It scales the domain of values from 0 to 100 and changes it to pixels based on the svg height. (line 3 and 10)
  • Draw the axis. Use d3.axisBottom for the x-axis (line 8). Use d3.axisLeft for the y-axis (line 15).
  • Using d3.line and the scales we created earlier to draw the line on the chart. (line 17)
  • Using d3.area create the shading on the line chart to diplay the high and low range of the dataset. (line 21)
 

 




 

 




 

 



 




const group = d3.group(data,d=>d.region)

const x = d3.scaleTime()
    .domain(d3.extent(data,d=>formatYear(d.year)))
    .range([margin.left,width-margin.right])

const xAxis = g => g.attr("transform",`translate(0,${height-margin.bottom})`)
    .call(d3.axisBottom(x))

const y = d3.scaleLinear()
    .domain([0,100])
    .range([height-margin.bottom,margin.top])

const yAxis = g => g.attr("transform", `translate(${margin.left},0)`)
    .call(d3.axisLeft(y))

const line = d3.line()
    .x(d=>x(formatYear(d.year)))
    .y(d=>y(d.median))

const area = d3.area()
    .x(d=>x(formatYear(d.year)))
    .y0(d=>y(d.high))
    .y1(d=>y(d.low))

Create a reusable chart.

  • Set up the svg inside the chart, to append a new svg each time the reusable chart function is called.
  • Annotate the chart. (line 11-37)
  • Draw the x-axis and y-axis by append a group element and calling the axis. (line 39-45)
  • Shade the line to indicate high and low values of the data. (line 47-52)
  • Draw the actual line. (line 54-59)
chart = (data,text)=> {
//create svg
    const svg = d3.select(".chart")
        .append("svg")
        .attr("width",width+margin.left+margin.right)
        .attr("height",height+margin.top+margin.bottom)
        .style("background-color","#3d0046")
        .append("g")
        .attr("transform",`translate(${margin.left},${margin.top})`)
//annotate the x -axis
    svg.append("text")
        .attr("x",width/2)
        .attr("y",height-8)
        .style("text-anchor","middle")
        .style("fill","#e3e077")
        .style("font-size","1.1em")
        .style("font-family","Times New Roman")
        .text("Year")
//annotate the y-axis
    svg.append("text")
        .attr("x",-height/2-105)
        .attr("y",15)
        .text("% of prevalence of anaemia")
        .attr("transform","rotate(-90)")
        .style("font-size","1.1em")
        .style("font-family","Times New Roman")
        .style("fill","#e3e077");
//create the chart title.   
    svg.append("text")
        .attr("x",width/2)
        .attr("y",20)
        .text(text)
        .style("text-anchor","middle")
        .style("font-size","1.5em")
        .style("fill","#e3e077")
        .style("font-family","Times New Roman")
        .style("font-weight","bold")
//draw the x-axis
    svg.append("g")
        .call(xAxis)
        .attr("color","#e3e077")
//draw the y-axis
    svg.append("g")
        .call(yAxis)
        .attr("color","#e3e077")
//shade the line
    svg.append("path")
        .datum(data)
        .attr("d",area)
        .attr("fill","#e3e077")
        .attr("stroke","none")
        .style("opacity",0.4)
//draw the line   
    svg.append("path")
        .datum(data)
        .attr("d",line)
        .attr("fill","none")
        .attr("stroke","#e3e077")
        .attr("stroke-width",2)
}
