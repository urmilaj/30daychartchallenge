These charts were made using a re-usable chart function. What's a re-usable chart function? The completed chart image above shows that each figure for the various animals is represented by a bar plot. Therefore each bar plot has a x-axis, y-axis, the bars, the title, all these elements are the same but the context is different, meaning the data is different. Therefore using a re-usable chart function comes in handy when we can just create the chart elements and then later call this function with the individual data and create the chart. Here's how to do this.
First, we create the reusable chart function.
Second, load the data files.
Finally, call the function with individual data.

Let's make the reusable chart function.
Let's name the function 'chart'. Now this chart function takes two parameters. The parameters are data and title. Now the data parameter will allow us to pass the individual csv files to the function. The title parameter allows us to give a title to our chart. Then lets return the values with these specific parameter to render the chart.

chart = (data,title) => {}
1

Inside the function lets now select our div which has a class name called chart and append an svg to it with our desired width and height and append a group element to this svg and position the group elements. Now every time we call the function a new svg element will be appended. Since we have five animal category data files, there will be a separate svg element for each animal.

chart = (data,title) => {

    const svg = d3.select(".chart")
        .append("svg")
        .attr("width",width+margin.left+margin.right)
        .attr("height",height+margin.top+margin.bottom)
        .style("background-color","#e6dee1")
        .style("margin-top","10px")
        .style("margin-bottom","20px")
        .append("g")
        .attr("transform",`translate(${margin.left},${margin.top})`)
}

Now lets make the title for each chart and an annotation. The chart title will be a re-usable element, for example, when a chart is called with the data for Sheeps and Goats , we can use the title parameter and write the title for the chart inside a string syntax and the svg will append the title to chart. We can also add a standard text element annotation showing that bar plot values are represented in lakhs, Indian Rupee which will reproduced in each chart. See the highlighted lines below.









 







 


chart = (data,title) => {

Now set up the scales and axis. Append the rect elements and axis ticks. Step one is done.

chart = (data,title) => {

    const x = d3.scaleBand()
        .domain(data.map(d => +d.year))
        .range([margin.left,width-margin.right])
        .padding(0.05)

    const xAxis = g => g.attr("transform",`translate(0,${height})`)
        .call(d3.axisBottom(x))

    const y = d3.scaleLinear()
        .domain([0,d3.max(data, d => +d.value)])
        .range([height,margin.top])

    const yAxis = g => g.attr("transform",`translate(${margin.left},0)`)
        .call(d3.axisLeft(y))
        .style("display","none")

    svg.append("g")
        .call(xAxis)
        .attr("stroke","#591d48")

    svg.append("g")
        .call(yAxis)

    svg.selectAll("rect")
        .data(data)
        .join("rect")
        .attr("x", d => x(+d.year))
        .attr("y", d => y(+d.value))
        .attr("width", x.bandwidth())
        .attr("height", d => height - y(+d.value))
        .attr("fill","#591d48")

    svg.append("g")
        .selectAll("text")
        .data(data.filter(d => d.value!=0))
        .join("text")
        .attr("x", d => x(d.year)+3)
        .attr("y",d => y(d.value)-5)
        .style("fill","black")
        .style("font-family","sans-serif")
        .style("font-size","0.75em")
        .text(d => "₹"+d.value)
}

Moving on to step two. Lets load the csv files. Since we have separate files for each animal we use Promise.all() and inside it an array of files, then we load the data which is a resolved promise. We take this resolved promise "data" and assign it separate variables where the variable names is the name of the animal which contains the corresponding data for it.

Promise.all([
    d3.csv("data/sheepAndGoats.csv"),

    let sheepAndgoats = data[0],

      horses = data[4]

Finally, call the chart function with these individual data and render the svg. Highlighted in the code lines below.














 
 
 
 
 


Promise.all([
    d3.csv("data/sheepAndGoats.csv"),
    d3.csv("data/bovine.csv"),
    d3.csv("data/swine.csv"),
    d3.csv("data/poultry.csv"),
    d3.csv("data/horses.csv")
]).then(function(data){
    let sheepAndgoats = data[0],
      bovine = data[1],
      swine = data[2],
      poultry = data[3],
      horses = data[4]

    chart(sheepAndgoats,"Sheep and Goats")
    chart(bovine,"Bovine")
    chart(swine,"Swine")
    chart(poultry,"Poultry")
    chart(horses,"Horses")
})
