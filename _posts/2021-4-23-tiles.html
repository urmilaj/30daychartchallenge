# African American Writers.

# Week 4: Timeseries. Day 23: Tiles.


Day 23's prompt was 'tiles'. I used the chart I made for Observable's black history month celebrations as the contribution for the topic. The chart I had made for black history month was in the tile format visually, each tile was associated with an african american author, the stroke colour differed based on the type of literary work. The tiles also had a fill pattern, the fill pattern indicated that the author had been recognised for their work with awards either during their lifetime or posthumously.

The final chart is below. Its an interactive visualization. It displays tooltips on hover and each individual tile will redirect to the literary work of the author on user click.

svg.call(pattern);

Each square is 60 pixels in size, each row has 10 squares and each column has 10 squares, so the whole size of each row is 600 pixels. This is specified below as variable "size", which is total size and "squareSize" which is size of each square.

Create the necessary scales for the chart.

  • the colour scale to distinguish each author.
  • A linear scale to create the tile format of the squares on the chart. The domain of the linear scale specifies that each row will have ten squares and the range specifies that each 60 pixel square should be placed between 0 pixels and 600 pixels.
const size = 600;
const squareSize = 60;
const color = ["#fed7af","#00ffff","#89065b","#ffea47","#e94f37","#6477b9","#fbd1fb","#ffa50a","#ffffff"];

const scale = d3.scaleLinear()
  .domain([0,10])
  .range([0,size]);

const colorScale = d3.scaleOrdinal()
  .domain(["activist","artist","author","journalist",
          "playwright","poet","publisher","scholar","sociologist"])
  .range(color);

This chart has a legend of circles on top to specify what the colour of each square means. This legend is created using Susie Lu's d3-legend library. To draw this legend use the code below.

  • Append a group element with a class name, and specify where this legend needs to be placed on the svg using transform and translate.
  • Using d3.legendColor specify the shape of legend as a circle, padding number places the circles at a distance from each other to prevent them from crowding and makes it legibile, d3-legend has a default orientation of vertical, to place the legend horizontally, specify it in orient. Finally call the legend on the svg to draw it.
svg.append("g")
  .attr("class", "legendOrdinal")
  .attr("transform", `translate(${width/2-300},20)`);

const legendOrdinal = d3.legendColor()
  .shape("circle")
  .shapePadding(55)
  .scale(colorScale)
  .orient('horizontal');

svg.select(".legendOrdinal")
  .call(legendOrdinal);

Now lets draw the tiles.

  • Append a group element.
  • Use a transform and translate to specify where the group tile elements will be placed.
  • Select all the rectangles, bind the data. Use enter to create a placeholder, append "a" which is a hyper link attribute which will redirect to an author's literary work available on the web when the user clicks on any square.
  • The attribute target blank specifies that external web link open in a new tab on the browser.
  • Append a rectangle, with class name.
  • Use attribute x and y to position the squares.
  • Specify width and height of each square.
  • Add event handlers to the squares.
  • When the user moves the mouse on any square, select "this" square (line 26), increase stroke-width to 3 pixels. Transition the tooltip we created earlier and change it's visibility from hidden to visible. Now append a heading element with the author name which is bound to the square and other details to be displayed in the tooltip div.
  • When the user moves the mouse away from the square hide the tooltip div (line 36-39).

























 









 
 
 
 

const waffle = svg.append("g")
  .attr("transform",`translate(${width/2-290},100)`)
  .selectAll("rect")
  .data(data)
  .enter()
  .append("a")
  .attr("xlink:href",d=>d.Work)
  .attr("target","_blank")
  .append("rect")
  .attr("class","waffle")
  .attr("x",(d,i)=>{
    const n = i % 10
    return scale(n)
  })
  .attr("y",(d,i)=>{
    const n = Math.floor(i/10)
    return scale(n)
  })
  .attr("width",squareSize-10)
  .attr("height",squareSize-10)
  .attr("stroke-width",1.5)
  .attr("stroke",d=>colorScale(d.Type))
  .style("fill",d=> {if (d.Awards!="-"){return pattern.url()}
                    else {return "transparent"}})
  .on("mousemove",function (event,d){
    d3.select(this).style("opacity",1).attr("stroke-width",3)
    
    authorTooltip.transition()
                  .duration(200)
                  .style("visibility","visible")

    authorTooltip.html("<h1>"+d.Name+"("+d.YearWritten+")"+"</h1><br/>"+d.About)
                  .style("transform","translateY(-55%)")  
                  .style("left",(event.x)+10+"px")
                  .style("top",(event.y)-15+"px")})
  .on("mouseout",function (event,d){
    d3.select(this).style("opacity",1).attr("stroke-width",2)
    authorTooltip.transition().style("visibility","hidden")
  })
