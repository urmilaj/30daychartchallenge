To create this tooltip lets append a div to the body, give it a class name and make its visibility hidden. Later we will add event listeners to the circles and make this tooltip appear on user interaction.

const alienTooltip = d3.select("body")
    .append("div")
    .attr("class","alienTooltip")
    .style("visibility","hidden")

The chart above has separate svg for each period. These svgs will be rendered using a reusable chart function. Here's the code below for the function.

chart = (data,height, chartTitle)=> {}
1
  • Inside the reusable chart function, lets set the svg with its width, height and other attributes. (line 3-9)
  • Set the chart title. (line 11-19)
  • Using d3.timeParse lets format the year column of the dataset to change it to appropriate date time format. (line 21)
  • Create the x-scale using d3.scaleTime which will help draw the x-axis later. (line 23-25)
  • Now let's create a placeholder for all the circles on the data using d3.selectAll, bind the data and join the circles. Set the radius, stroke, stroke-width and fill of the circles. Lets also attach event handlers to the circles, indicating that when a user hover over the circle we would like a tooltip to display details and when the user stops hovering, the tooltip disappears. (line 27-48)
  • When we created the circles above we only set the radius attribute of the circle, what about the position of each circle, i.e the attribute center-x (cx) and center-y (cy), well these attributed are set using d3.forceSimulation.
  • Using d3.forceSimulation and the scales created earlier lets set the circle position attributes. (line 50-53)
  • Call this simulation on the data, to place the circles in the right position when the simulation ticks. (line 56-58). Now when the simulation ends, call the tooltip function we created.
chart = (data,height, chartTitle)=> {

const svg = d3.select(".viz")
            .append("svg")
            .attr("width",width+margin.left+margin.right)
            .attr("height",height+margin.top+margin.bottom)
            .style("background-color","#51244f")
            .append("g")
            .attr("transform",`translate(${margin.left},${margin.top})`)

svg.append("text")
    .attr("x",width/2)
    .attr("y",30)
    .text(chartTitle)
    .style("text-anchor","middle")
    .style("font-size","1.3em")
    .style("fill","#eaaa87")
    .style("font-family","Times New Roman")
    .style("font-weight","bold")

const formatYear = d3.timeParse("%Y")

const x = d3.scaleTime()
    .domain(d3.extent(data,d=>formatYear(d.Year)))
    .range([margin.left,width-margin.right])

const circle = svg.selectAll("circle")
    .data(data)
    .join("circle")
    .attr("r",radius)
    .attr("fill",d=>d.Abduction==1?"white":"none")
    .attr("stroke","#eaaa87")
    .attr("stroke-width",1.8)
    .on("mousemove",function (event,d){
    d3.select(this).style("opacity",1).attr("stroke-width",3)
    
    alienTooltip.transition()
                .duration(200)
                .style("visibility","visible")

    alienTooltip.html("<h3>"+d.Name+", "+d.Location+"</h3>")
                .style("transform","translateY(-55%)")  
                .style("left",(event.x)+10+"px")
                .style("top",(event.y)-15+"px")})
.on("mouseout",function (event,d){
    d3.select(this).style("opacity",1).attr("stroke-width",2)
    alienTooltip.transition().style("visibility","hidden")
})

const simulation = d3.forceSimulation()
    .force("x",d3.forceX(d=>x(formatYear(d.Year))).strength(1.8))
    .force("y",d3.forceY(height/2))
    .force("collide",d3.forceCollide(radius+padding).strength(2))
                

simulation.nodes(data)
    .on("tick",d=>circle.attr("cx",(d,i)=>d.x)
                        .attr("cy",(d,i)=>d.y))
    .on("end",tooltip)

This tooltip is not the same as the hover tooltip. Look at this image of the chart. This tooltip function is actually the annotations on the chart made using Susie Lu's d3-annotation library. This tooltip function is actually the annotations displayed.

  • To create the tooltip function or to be more specific the annotations, the code is as follows.
  • Create the function, set a labels variable and filter the data where the abductions column have a value equal to 1. Now map this information and set the x, y, dx and dy values. These values will be used to position the annotations. The x and y values will be derived from the d.x and d.y which is calculated by d3.forceSimulation. Now the dx and dy values in the dataset is what I have manually added to the dataset. These values will help to place the annotations in a legible manner. Note and colour is what the d3-annotation library uses to place title and label text inside the annotation, colour also helps set the colour of the svg annotations.
function tooltip(d) {
    let labels = data.filter(d=>d.Abduction==1).map((d,i)=>{
    return {
        x:d.x,
        y:d.y,
        dx:d.dx,
        dy:d.dy,
        color:"#eaaa87",
        note:{
        label:d.Name,
    }
    }
    })

    const type = d3.annotationCustomType(
    d3.annotationCalloutElbow, 
    {"className":"custom",
        "connector":{"end":"dot"},
        "note":{"lineType":"horizontal",
        "align":"middle"}})

    let makeAnnot = d3.annotation()
        .type(type)
        .textWrap(120)
        .annotations(labels)

svg.append("g")
    .call(makeAnnot)
}
