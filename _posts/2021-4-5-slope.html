# Pyramids of Egypt

# Week 1: Comparisons. Day 5: Slope.


The topic for day five was slope. I took the topic quite literally and visualized the pyramids of Egypt and found out some interesting information. The pyramids of Egypt look like they all have the same slope but apparently this is not true and each pyramid have slopes unique to them. Egyptians measured slope in seked, an egyptian length of measure also known as royal cubit.

Load the textures library and d3-annotation library, which will later be used to add textures to your triangle to give a pyramid effect and annotate the chart.

Draw the triangles, make the fill for the triangles using textures declared earlier.

// Draw the path for the triangles.
svg.append("g")
    .attr("transform",`translate(${width/2},100)`)
    .selectAll("path")
    .data(data)
    .join("path")
    .attr("transform", (d,i) => `translate(0,${i*250})`)
    .attr("d",triangle)
    .style("fill",texture.url())
    .style("stroke","#faac75")
    .style("stroke-width",3)
